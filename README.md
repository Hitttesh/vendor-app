# Vendor Assessment Management System

A full-stack Vendor Portal that allows vendors to create assessments, upload candidate resumes (via Firebase), track candidate statuses, and manage their account.  
Built using **FastAPI**, **PostgreSQL**, **SQLAlchemy**, **React**, **Tailwind CSS**, and **Firebase Storage**.
